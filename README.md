# Qalagan landing

Статический лендинг (HTML/CSS/JS) для сервиса персональных страниц записи.

## Структура
- `index.html` — структура страницы и секции лендинга.
- `styles.css` — стили, сетки, адаптивность.
- `script.js` — локализация, ротаторы демо-карточек, UI-логика.
- `translations.json` — тексты интерфейса (RU/KK/EN).
- `specialists.json` — демо-данные карточек в hero/showcase.

## Локальный запуск
```bash
python -m http.server 4173
```

Откройте `http://localhost:4173`.
