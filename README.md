# Qalagan

Qalagan — это сервис онлайн-записи на услуги через одну ссылку.

Клиенты могут записываться через Instagram, WhatsApp, Telegram и другие социальные сети, не переписываясь в Direct.

## MVP включает

- страницу записи
- выбор услуги
- выбор дня
- выбор времени
- предоплату
- быстрые окна
- календарь
- аналитику

## Статус

MVP prototype in progress.
